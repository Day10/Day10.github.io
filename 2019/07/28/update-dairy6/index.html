升级日记（六）

发表于 2019-07-28

这周看了一个欧美的综艺《粉雄救兵》，是五个gay从发型，穿搭，文化，饮食和室内装修这几个方面去帮助一个人改变，我其实看过很多类似这种素人改造的视频，但是这个综艺真的非常让人感动，因为他们是真正思考了这个人的地位，年龄来帮助他改变外貌以及内心的，虽然只是个综艺，但是能看出每个经过改变的人，精神状态都比之前好了很多。我很羡慕这样的工作，带给人们幸福，带给人们美好的工作实在太棒了。
这周看了奥巴马的一些演讲，觉得这个人真的非常有人格魅力，而且非常风趣，所以下载了电子书，准备看一看他的演讲。
这周看了两个让我引以为戒的人，一个是《魔道祖师》里面的苏涉，一个是《非自然死亡》里面第五集的凶手。前者在最后想折辱蓝忘机，只是因为看不惯“他那副总觉得自己很了不起的样子”，后者杀死了一个无辜的女孩，只是因为“为什么那样的人，会比我幸福”这种理由，我觉得世界上真的有这种无缘无故的恨，这种恨大约源于嫉妒和自己自卑的心灵。其实曾经，我也嫉妒过别人，这种感觉真的非常非常难受，嫉妒这种心情，不仅丑陋而且真的让人很痛苦，此生都不想再感受了，所以我想，必须要努力，必须要有自己的目标，只有这样，才能坚定，才能看到别人的成功或者失败都不会动摇，也不会深陷嫉妒的泥沼。
因为这周还是没有想到什么素材，所以找到了很久很久以前的，看过的杂志，叫《巧克力季节》，这本杂志现在已经买不到了，只能在网上看一些文章。里面有一篇同人文，我至今都影响深刻，叫《关于一个番茄》，是赤西仁和山下智久的同人文，文章的风格非常特别，所以下周想试着模仿这种风格，写一写。
