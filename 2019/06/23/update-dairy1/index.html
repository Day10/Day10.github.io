升级日记（一）

发表于 2019-06-23 | 分类于 Update Dairy

有一句话说的好，“人是不能太闲的,闲久了就习惯了,努力一下就以为在拼命”。这周隐隐有这种自己在拼命的感觉，但是仔细一想，其实做的还远远不够。

说起来这周我读了一本书《中华的另一种可能-魏晋风流》，简直沉迷其中不可自拔。读完一半了，脑子里只有一个感想：原来古人都这么有意思的吗？然后返回头去读了小时候学过的《兰亭集序》和《归去来兮辞》，只觉得这些年的语文都白学了。回想当年学古文，要背文章，背翻译，背古文下面满满的注释，就让人很头疼了，完全没有心情去体会作者的心情。但是此时再读，只觉得有趣，只觉得即使跨域了千年的时光，人的心情都是共通的。

王羲之跟友人相聚，会抒发快乐的情绪，又觉得时光易逝，天下无不散之宴席。陶渊明，我觉得和我们现在有何不同，觉得上司是“山野小民”，就潇洒的辞职了。辞职后写下了《归去来兮辞》，告诉大家，我辞官是因为我工作是为了口腹之欲，和我自己的内心不符，而且我喜欢田园的安逸生活。和我们所说的“世界这么大，我想去看看”很像。这么一想，只觉得古文着实妙趣横生！
